# Blog-PHP-JavaScript

Sitio web con 3 páginas .php según la estructura adjunta, la primera página recoge las novedades de un fichero json ya creado que contiene 6 registro. Los destacados son las 3 primeras imágenes que ha subido el usuario en la página de blog. En la segunda página los datos los saca desde la tabla ARTICULOS, en la que hay introducidos previamente 5 registros, pero solo muestra 3 y la página de BLOG, recoge los datos de la tabla BLOG, en la que los usuarios registrados pueden insertar registros desde es misma página si se han logado en la primera.
